4.1 – Home Network Basics

4.1.1 Video — Typical Home Network Setup

  • Most home networks consist of at least two separate networks
  • The first network is the public network that is coming in from the service provider, usually a cable or DSL connection.
  • The DSL provider or cable provider network does not use the same protocols and transmission methods as does an ethernet network
  • So one thing you are going to see in almost every home network is a modem
  • The modem converts the signals coming from the provider network
  • The modem has an input for the Coaxial cable and it also has an output port that goes to the local network
  • Almost all home networking devices provide wireless connectivity
  • Modems will have internet port and LAN connectivity ports, these ports are usually switched ports
  • LAN ports are used to connect devices within the local network
  • WAN ports are used to connect to the internet
  • Most modems have built in wireless access point called a Router
  • The reason why they are called routers is that the ports are separated into two separate networks
  • The first network is called the local network, which can be wired (plug into an Ethernet port) or wireless (connect via Wi-Fi).
  • The second network is the wide area network (WAN), which connects to the internet.
